# 12_image_resize

## Usage

 - Run `python image_resize.py <path_to_original_file> <options>`

## Options:

name | type | key |
--- | --- | --- |
**Scale** | float | `-s, --scale`
**Width** | int | `-w, --width`
**Height** | int | `-h, --height`
**Output** | string | `-o, --output`

Note that you can't use both scale and width / height options
